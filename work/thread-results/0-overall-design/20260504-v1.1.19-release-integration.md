`packages/release-chain/v1.1.19/app-debug-20260504-v1.1.19.apk`

## 本次必须进新 APK 的原因

`v1.1.18` debug APK 中存在休息通知“打开训练/返回训练”可能误进入 debug 自测页的问题。该问题影响用户从通知回到训练页的关键路径，因此不能继续沿用 `v1.1.18` 发布包。

## 已确认修复点

- `RestTimerAlert.kt` 通知返回显式启动 `MainActivity`。
- `RestTimerService.kt` 运行中通知返回显式启动 `MainActivity`。
- `RestTimerAlarmScheduler.kt` alarm-clock show intent 显式启动 `MainActivity`。
- `app/src/debug/AndroidManifest.xml` 中 `RestReminderSelfTestActivity` 不再声明 `MAIN` / `LAUNCHER`。
